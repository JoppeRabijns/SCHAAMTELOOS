## 💻 Tech Stack

**Client:** React, Redux, GSAP, Socket.io

**Server:** Node, Express, NexRender, Twillio, Socket.io


## 🚀 Lokaal starten

Clone the project

```bash
  git clone https://github.com/JoppeRabijns/SCHAAMTELOOS.git
```

Go to the project directory

```bash
  cd FRONTEND
```

Install dependencies

```bash
  npm install
```

Start the server

```bash
  npm start
```
